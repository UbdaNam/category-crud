# 📖 Category-CRUD <a name="about-project">

Hi there! Welcome to Category-CRUD</a>

Category CRUD is a dynamic web application that enables users to view and manage a hierarchy of categories and their corresponding sub-categories. This intuitive platform offers convenient Add, Edit, and Delete functionalities for each category, allowing for easy organization and customization.

### Key Features <a name="key-features"></a>

- **View category hierarchy**
- **Add sub-categories**
- **Edit category**
- **Delete category**
- **Zoom category tree**
- **Drag category tree**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to install Node.js, Web browser, and code editor (recommended: Visual studio).

### Setup

Clone this repository to your desired folder:

```snippet
  git clone https://github.com/UbdaNam/category-crud.git
```

### Install

```snippet
  npm install
```

Install this project with: Any code editor which you are comfortable with

### Usage

In the project directory, you can run:

```snippet
  npm run dev
```
